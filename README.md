Abetone Trail Park PWA
======================


Italiano
--------

Questo è il porting in PWA dell'applicazione Abetone Trail Park, originariamente scritta in Cordova.

La ridizione di questa App in formato PWA permette agli utenti il suo uso senza effettuare una installazione dagli store Google ed Apple, aprendone l'uso anche ad altri dispositivi non specifici.

Per quanto riguarda l'installazioen della presente App in Android (ma anche sistemi Windows e Linux) è veramente ottimale con l'uso dell'install button previsto dallo standard PWA. Un po' più problematica è l'interazione con gli utenti Apple (iPhone, iPad, ecc.) i quali hanno, per via della piattaforma che non suporta pianamente lo standard, più difficoltà a completare l'installazione della stessa.

Alcuni file Cordova sono rimasti nell'applicazione per semplificare la possibilità di ricompilare l'applicazione in App Cordova.

Le mappe sono create con il Mobile Atlas Creator 2.3.1.

Su Windows, per compilare l'applicazione nell folder "build" è necessario avere installato node, npm e workbox ed eseguire build.bat.
Su Linux è necessario scrivere uno shellscript che effettui le stesse funzioni del file batch di Windows.

L'applicazione è in funzione qui: https://www.abetonetrailpark.com/app/

//

English
-------

This is the porting to PWA of the Abetone Trail Partk application, which was previously written for Cordova.

The republishing of this App in PWA format allow users to use it without installing it from Google or Apple stores and enabling its use also by other devices.

About the installation of this App in Android (but also on Windows and Linux systems) it is very optimal with the use of the install button provided by the PWA standard. iPhone and iPad users must pay much more attention during the install phase because the lack of support by Apple to the install button functionality of the PWA standard.

Some Cordova related files are kept in the application to simplify future recompile it to a Cordova App.

Maps are created with Mobile Atlas Creator 2.3.1.

On Windows, to compile the app inside the "build" folder, just install node, npm and workbox and run build.bat.
On Linux you have to provide a shellscript to perform the same operations of the windows batch file.

The runtime application is available from: https://www.abetonetrailpark.com/app/

//
